# Nestlé CommHub

Nestlé CommHub is a unified B2B platform for streamlined communication, promotion management, and stock optimization between Nestlé HQ and its retail partners.

## 📄 Documentation
- **[System Logic & Architecture Report](./SYSTEM_LOGIC.md)**: Detailed breakdown of user roles, business flows (Promotions, Tickets, Stock), and technical stack.
- **[Test Credentials](./credentials.md)**: Accounts for testing role-based features.

## 🚀 Quick Start
### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd app
npm install
npm run dev
```

### Seeding Data
```bash
cd backend
node scripts/seed-data.js
```
